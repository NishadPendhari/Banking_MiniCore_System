# Banking_MiniCore_System

## Description
This is a simple Spring Boot project for banking operations like deposit, transfer, and audit logging.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- H2 Database

## Features
- Deposit money
- Transfer money
- Audit logging
- REST APIs

## How to Run
1. Clone the project
2. Open in IDE
3. Run main class
4. Use Postman to test APIs
